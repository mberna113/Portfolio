About Me

My journey into spatial problem-solving began in the U.S. Army as a Cavalry Scout, where terrain, navigation, and geolocation were critical to every mission. That passion for place-based precision grew while instructing at the Northern Warfare Training Center, where I taught Cold Weather Warfare and Military Mountaineering. In those environments, knowing exactly where you were could mean the difference between success and failure.

Later, I discovered the field of Geographic Information Systems (GIS) and realized there was a whole discipline devoted to answering spatial questions. Since then, I’ve been combining my love of the outdoors with solving real-world problems using maps, data, and analysis.

In the age of big data, I believe we have more tools and knowledge than ever to help improve—and even save—lives through spatial insight. My work blends hands-on experience with cutting-edge GIS tools to do just that.

Projects

Amherst GID Analysis

This project analyzed the Amherst General Improvement District using ArcGIS Online and Experience Builder. The interactive web app visualizes the GID boundary, tax zones, and collected resident feedback to support a transparent review of whether the added taxation aligns with local infrastructure or community benefit. This project involved spatial joins, demographic overlays, and custom experience design.

Land Use Change Detection with GEE

This project explores land cover change using cloud-based remote sensing tools in Google Earth Engine (GEE). It highlights how dynamic area selection and automated classification can lower the barrier for non-experts to explore meaningful spatial change over time. The briefing demonstrates how land use insights can support sustainable decisions in urban planning, ecology, and agriculture.

  • Applied automated classification workflows based on current scientific studies and remote sensing methodologies.

Real-Time Flood Monitoring Dashboard

This project showcases a real-time flood monitoring system built using ArcGIS Experience Builder and a custom ETL pipeline. The system integrates live weather and stream gage data with Python and PowerShell scripts running on a virtual server. This dashboard was developed to demonstrate how automation and spatial data can be combined to improve emergency awareness and decision-making.
